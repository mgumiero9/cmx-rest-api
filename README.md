# Table of contents

This project contains the main REST API for Comex test.

Technologies:

- Java
- SpringBoot
- REST API
- Swagger Open API
- JUnit

## Requirements

* JDK 8
* Your favorite IDE
 ( Se recomienda instalar la [IntelliJ IDEA](https://www.jetbrains.com/idea/download/) ).

## Development Quick Start

```bash
git clone https://github.com/mgumiero9/cmx-rest-api.git
cd cmx-rest-api
./gradlew clean bootRun -Dspring.profiles.active=dev
```

*If you prefer, you can configure your Gradle with **bootRun** Task and run it from IDE*

**The API listen at port 8080**.

### IntelliJ IDEA Setup

* Configurar default Project SDK (en el Welcome Screen --> Configure --> Projects Default --> Project Structure)
* Import project (Seleccionar el build.gradle)
* Usar defaults (gradle wrapper, etc)
* Run --> Edit Run Configurations
* Add Gradle Task (ponerle nombre COMEX por ejemplo)
